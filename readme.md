## 安装

1. 克隆此仓库：
   ```bash
   git clone https: //github.com/yourusername/your-repo.git
   ```
2. 进入项目目录：
   ```bash
   cd your-repo
   ```
3. 安装依赖：
   ```bash
   npm install
   ```

## 使用

1. 启动服务器：
   ```bash
   npm start
   ```
2. 打开浏览器并访问 `http: //localhost:3000`。

## 功能

- 搜索功能：用户可以通过输入关键字搜索资产。
- 无限滚动：当用户滚动到页面底部时，自动加载更多资产。
- 占位符图像：在图像加载之前显示占位符。
- 复制 Token ID：用户可以通过点击按钮复制 Token ID。
- 防抖动输入：减少输入时的 UI 卡顿现象。
